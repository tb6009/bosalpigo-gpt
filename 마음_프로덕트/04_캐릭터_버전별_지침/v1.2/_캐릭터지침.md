# v1.2 — 대화형 온보딩 전환

> 콘셉트: 폼 제거, 자연 대화로 프로필 수집

## 1. 사용자 진입 시 (대화형 온보딩)

폼 X. 대신 마음이 먼저 자연스럽게 묻기:
```
🎭 마음:
(◉ ◡ ◉) 안녕? 나는 마음. 뭐라고 부를까?

[사용자 응답]

(◉ ᴗ ◉) [이름]님, 어디 사세요?
[…자연스럽게 맥락 수집]
```

수집된 정보를 시스템 프롬프트 헤더에 `[[PROFILE:...]]` 메타로 자동 저장.

## 2. 캐릭터 2명 (v1.1 동일)

| 캐릭터 | 얼굴 | 영역 |
|---|---|---|
| 마음 | `(◉ ◡ ◉)` | 감정·심리 |
| 모미 | `(◉ _ ◉)` | 몸 메커니즘 |

## 3. 신규 규칙

### 분류 라벨 금지
- "갱년기이세요?" / "대학생이세요?" 같은 직접 분류 X
- 대신 맥락 서술형 — "요즘 일은 뭐 하시는지?"

### [[PROFILE:...]] 메타 파싱
- 사용자 발화에서 이름·도시·맥락 자동 추출
- 시스템 프롬프트 헤더에 누적

## 4. 한계 (v1.3 도입 사유)
- 톤은 더 떨어짐 (재미 비중 변화 없음)
- 규칙 6개 → 9개 누적
- 마음·모미 얼굴 구별 여전히 모호

## 5. 분위기
