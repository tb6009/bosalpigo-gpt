## 평가 대상

| 버전 | 캐릭터 구성 | 평가 대상 |
|---|---|---|
| v1.3 | 마음 + 모미 | 마음, 모미 |
| v1.4 | 마음 + 모미 | 마음, 모미 |
| v1.5 | 마음 + 모미 + 요기 + 숨이 | **마음, 모미만** (요기·숨이는 측정 외) |

**핵심 조건**
- 두 캐릭터가 같은 응답에 동시 등장하지만 **한 발화에 한 캐릭터만 말함** (만담 구조)
- 따라서 평가 단위는 (1) 캐릭터별 발화 단위 (2) 응답 전체 단위 두 층으로 분리
