## ⚠️ 평가 한계 (먼저 읽을 것)

1. **풀 시스템 프롬프트 부재**: 현재 저장된 `_캐릭터지침.md`는 콘셉트 요약본 (1.2~2.2KB). 기존 리포트가 측정한 실제 시스템 프롬프트(3.4~6.6KB)와 1:1 비교 불가. → 본 리포트는 **콘셉트·정책 지침 기준**으로 재측정.
2. **v1.5는 마음+모미만 비교**: 사용자 요청에 따라 요기·숨이는 본 비교에서 제외. 단 v1.5 안전성·측정 점수는 4-캐릭터 위계 전제하에서 작동하므로, 듀오만 추출 시 일부 안전 기능이 "위임"으로 표현됨에 주의.
3. **사용자 체감 데이터 부재**: Fidelity Score 100점 중 "사용자 체감(20점)"은 Phase 3 측정 미완. 본 리포트는 추정치 표시.
4. **시나리오 응답 시뮬레이션**: 실측 LLM 응답이 아닌, 시스템 프롬프트·지침 기반 예상 응답. v1.0~v1.3은 기존 리포트 인용, v1.4~v1.5는 신규 추정.

## 🎭 2. 버전별 캐릭터 DNA (압축)

### v1.1 — "듀오 탄생, MI 도입"
- 마음+모미 분리, MI/OARS/DARN-CAT
- 폼 온보딩 (닉네임·도시·페르소나)

- 분위기: *"대학 상담센터의 쿨한 상담사 듀오"*

### v1.2 — "대화형 온보딩 전환"
- 폼 제거 → 자연스러운 질문으로 수집
- "갱년기이세요?" 류 분류 라벨 금지 추가
- 톤은 v1.1보다 더 점잖음

### v1.3 — "주제 경계 + 이모티콘 절대 규칙"
- 재정·진로·관계 등 6개 금지 주제
- 이모티콘 "어기면 실격" + `fixCharFaces()` 사후 교정
- 분위기: *"규칙집 들고 온 진지한 상담사"* — **톤 최저**

### v1.4 — "톤 복구 + 만담 + 음성"
- v1.0 톤 + v1.3 안전 + v1.1 듀오 구조 결합 시도
- 마음 = 네모 안경 `【⊡ ◡ ⊡】`로 모미와 시각 차별화
- 만담 구조 (한 캐릭터 응답 후 상대 한 줄 추가)
- TTS·STT, Export 추가
- "MI 프로토콜 5단계" → "듣고, 인정하고, 되비추고, 정리해줘"로 **용어 숨기기**
- 분위기: *"동네 친구 둘이 같이 떠드는" + 안전망*

### v1.5 (마음+모미 추출) — "측정 + 종단 추적"
- **마음**: 코미디언 7 + 츤데레 3, 측정 도구 4종 (PHQ, GAD, PSS, BRS)
- **모미**: 친언니 50 + 트레이너 50, 측정 도구 4종 (PSQI, FSS, EAT-26, IPAQ)
- L1-a 듀오 모드 (기본 진입 형태)
- **MI Talk Time 원칙 도입** (캐릭터 합산 ≤ 사용자 직전 발화 × 1.5)
- **만담 횟수 동적화** (사용자 1발화에 캐릭터 4발화 이상 절대 금지)
- 종단 변화 추적 ("전에 이런 얘기 했는데 지금은 어때요?")
- 디스클레이머 의무
- ⚠️ 듀오만 보면 위기·트라우마 처리는 **숨이로 위임**되어 있어 듀오 자체의 안전 응답 능력은 v1.4와 유사

## 🚧 7. v1.5 듀오만 추출 시 한계

요청에 따라 **마음+모미만 비교**한 결과 다음 항목이 손실됨:

| 손실된 능력 | v1.5 전체에서 담당 | 듀오만 시 처리 |
|---|---|---|
| 위기 표준 응답 (109/1577) | 숨이 | 마음 P-2 한계 인정으로 위임만 |
| 트라우마 인폼드 케어 (TCTSY 5원칙) | 숨이 | 처리 불가 — 위임 |
| 신체 동작 안내 (HRV 호흡, 요가) | 요기 | 모미가 "설명"만, 안내 X |
| 화병 (한국 문화 특이성) | 숨이 | 모미가 위임 |
| 과호흡·공황 안정화 | 숨이+요기 | 마음이 안정 시도, 한계 명확 |

**결론**: v1.5의 진짜 평가는 4-캐릭터 위계 전체에서 가능. 듀오 추출은 v1.4와의 직접 비교(만담 구조·MI Talk Time·측정 도구 도입 효과)에 적합하지만, **안전성·트라우마·위기**는 별도 평가 필요.

## ✅ 8. 결론·권고

### 8-1. 정량 결론
- **v1.5 듀오 = 87점** (v1.4 대비 +4)
- **v1.5 전체 = 91점** (v1.4 대비 +8)

- **v1.4의 "톤 복구"는 성공** (v1.3 1:16 규칙 과잉 → v1.4 만담 구조)
- **v1.5의 "측정 + 종단 추적" 전환은 임상 연계의 분기점** — 단순 챗봇에서 자기 인식 보조 도구로 진화
- 듀오만으로도 MI Talk Time·동적 만담·종단 추적의 가치가 분명히 드러남
