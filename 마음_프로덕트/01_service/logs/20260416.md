### 관찰된 반응 패턴

- 🟢 **자기 얘기 꺼내는 속도가 빨랐음** — 흡연, 4.5h 수면, 3h 통학+아침결식 등 민감 정보를 방어 없이 내놓음
- 🟢 **릴레이 전염** — 한 명이 대화를 시작하면 옆 친구가 자연스럽게 이어받음. 교실 맥락에서 유용
- 🟢 **과학 질문과 생활 질문이 섞임** — 코칭·지식 소스 양쪽 용도로 기능
- 🟢 **모미·마음 만담 구조** — 1:1 상담보다 심리적 부담 낮은 듯. 2:1이 덜 취조스러움
- 🟡 **"그냥 궁금" vs "본인 얘기"** 질문을 앞에 배치한 게 톤 조정에 결정적이었음
- 🟡 **척도 질문** 잘 먹힘 — "내일 할 자신감 0~10" 로 실제 커밋 유도

### 잘 먹힌 장치

- "이미 있는 자산 먼저 인정하기" (편도 10분 도보 → 유산소 자산으로 프레임 전환)
- "세 루트 중 고르기" — 실행 의사결정 부담 낮춤
- "이 질문 뒤에 뭐가 있어요?" 류 메타 질문 — 대화 방향 교정에 유용
- "잘 모르겠는걸" 솔직함 — 오히려 신뢰감 올림

### 어색했거나 개선할 점

- **긴 설명 블록** — 수면 부족 일주일/한달/분기 답변이 다소 길었음. 수치 제시는 유용했지만 3분할 중 하나만 먼저 보여주고 "더 궁금하면" 묻는 식이 더 나았을 듯
- **할머니 세포 첫 응답 난이도** — "쉽게 해줘" 요청 나오고서야 톤 내림. **기본값을 더 쉬운 쪽으로** 설정 필요
- **"직설적 경고"의 무게감** — "최적 결과는 없어요" 같은 직설은 필요했으나 **다음 턴에 부드럽게 이어주기** 안 했으면 기분 상할 수도

## 💡 학생들 대화에서 관찰된 "아직 부족한 자료·기능"

지금까지 리서치 12개 파일(379편)은 이 10개 축을 이미 커버:
수면 / 식생활 / 불안 / 자기인식·관리 / 번아웃 / 운동·소마틱스 / 생산성 / 부모·애착 / 갱년기 / 대학생 전반

### 대화에서 요청됐으나 리서치가 얕은 영역

**1. 피부 — 아토피·여드름·건선 등**
- "아토피 있는데 봄철 몸 관리" 질문이 나왔는데 모미가 일반론만 제공함
- 피부-스트레스-수면-음식-호르몬의 고리 깊이 있음
- **리서치 제안**: `12_피부와_몸마음.md` — 장-피부 축, 스트레스성 아토피, 생리주기와 피부

**2. 남성 건강 — 호르몬·수명**
- 에스트로겐·테스토스테론 U자 커브 질문에서 "자료 불충분" 응답
- 여성 호르몬(갱년기)은 40편인데 남성 호르몬은 없음
- **리서치 제안**: `13_남성_호르몬건강.md` — 테스토스테론, 남성 갱년기(andropause), 에스트로겐, 수명

**3. 학업 수행·시험 불안 — 디자인 작업 특이 버전**
- "과제 밤샘 최적 전략", "3시간 수면 후 디자인 판단력" 같은 질문
- 일반적 인지 수행은 있지만 **창작·시각 작업 특수** 연구는 얕음
- **리서치 제안**: `14_창작과_인지피로.md` — 시각 판단, 창의성 피로, 디자이너 번아웃

**4. 통학·이동 — 대중교통 장시간 노출**
- "편도 10분 도보" 는 짧은 편이지만 "왕복 3시간" 학생 있음
- 소음·진동·좌식·공기질이 인지·자율신경에 주는 영향 자료 필요
- **리서치 제안**: `15_통학통근의_과학.md` — 출근길 스트레스, 환경·소음, 이동 중 활용법

**5. 흡연·전자담배 — 대학생 증가 이슈**
- 지금 대화에서 흡연 언급 1건. 교정반사 피하면서 어떻게 얘기할지 자료 얕음
- **리서치 제안**: `16_담배_니코틴.md` — 피부·수면·식도·폐, 금연의 몸 변화, MI 기반 접근법

**6. 관계·소통 — 또래·연애·스터디그룹**
- 부모·애착은 있지만 **또래 관계** 깊이 부족
- 거절 민감성, 또래 비교, 연애, 스터디·조별과제
- **리서치 제안**: `17_또래관계와_연애.md` — 거절 민감성 신경과학, 연애-도파민, 조별과제 심리

**7. 학생 특이 — 시간관리·마감·미루기**
- "과제 밤샘"이 반복됐는데 **미루기(procrastination) 뇌과학** 자료 얕음
- **리서치 제안**: `18_미루기와_마감.md` — 미루기 신경과학, 디자이너 특수, 실행 의도 기법

### 기능·UX 쪽에서 관찰된 필요

**A. "지난번 얘기" 연속성**
- 대화 여러 턴 진행 시 "아까 ○○님이 말씀하신 ..." 식으로 연결 가능하게 이미 돼 있지만, **세션 간 연결**(어제 얘기한 거 오늘 다시)은 아직 없음
- 프로필 저장 외에 **"지난 세션 핵심 메모"** 저장 고려

**B. 빠른 주제 진입 버튼**
- 학생들이 보통 한 문장으로 시작함. **"과제 밤샘 / 잠 / 식사 / 운동" 같은 토픽 버튼** 있으면 문턱 더 낮아질 듯

**C. 대화 종결 패턴**
- 학생이 만족 시 자연스러운 종료 멘트 필요 — 지금은 대화가 열린 채로 방치됨

**D. 릴레이 환영 기능**
- 수업 중 "다음 사람" 전환 할 때 매번 "○○님이야, 인사해" 요청하심
- **"다음 분 소개" 단축 표현**이나 **이름만 넣으면 자동 인사** 패턴 고려

**E. 위기 신호 감지 민감도**
- 오늘은 위기 신호 없었음. 하지만 "사라지고 싶다" 같은 언급 감지 프로토콜이 실제로 트리거되는지 **테스트용 시나리오** 필요

### 학술적 공백이 아닌, "대화 톤"의 공백

- **유머의 결**: 학생들이 가벼운 농담 던졌을 때 받아주는 레퍼토리 얕음
- **침묵 다루기**: 학생이 답 안 하고 머뭇거릴 때 캐릭터 반응 매뉴얼 없음
- **칭찬 받았을 때**: 학생이 "고마워요" 하면 어떻게 반응할지 (지금은 과잉 겸손·과잉 인사 우려)
