## 4. 규칙·금지 명령 비교

### v1.3 — "16개 규칙의 시대"

```
0순위 톤 탄력성 (5개 sub-rule)
1순위 정확성 (4개 sub-rule)
2순위 행동 변화 (3개 sub-rule)
+ 캐릭터별 역할 (3개)
+ 등장 규칙 (4개)
+ 주제 경계 (6개 + 4단계 처리 + 예시)
+ 이모티콘 절대 규칙 (6개)
+ MI 프로토콜 6단계
+ 금지어 8개 ("힘내세요", "이해합니다" 등)
+ 위기 프로토콜 4단계
+ 트라우마 경계 3개

총: ~16 그룹의 명시적 규칙
```

### v1.4 — "5규칙의 시대"

```
1. 재미가 기본값
2. 모르면 "잘 모르겠는걸"
3. 설교 금지
4. 작은 실천으로 연결
5. 위기 신호 시 농담 OFF + 1393/1577

+ 만담 구조 (3가지)
+ 이모티콘 분리 규칙 (1줄)
+ 응답 형식 (3~6줄)

총: 5 핵심 + 3 구조 = 8 그룹
```

### v1.5β — "위계 + 정밀의 시대"

```
4 절대 원칙 (의학적·위기·헤더·한계)
4 캐릭터 정의 (각 6~8 sub-rule)
3 위계 + 위임 멘트 (4개)
위기 프로토콜 (1 표준 응답 + 동의 우회)
측정 도구 운영 (4 sub-rule)
한계 인정 5요소
응답 형식 (헤더 + 8-1 만담 횟수 + 8-2 Talk Time + 8-3 발화 상한 + 8-4 끼어들 공간)
9 절대 금지 (전 캐릭터 공통)

총: 11 그룹, 그러나 각 그룹 내부가 정밀
```

## 9. 학술적 근거 인용 (v1.5β만)

v1.3, v1.4에는 학술 근거 인용 **없음**. v1.5β에서 시스템 프롬프트 자체에 인용 등장:

| 인용 | 사용 위치 |
|---|---|
| **Laborde 2022 RCT 223** | 요기 호흡 카운트 표준 (흡 4/호 6, 분당 6회) |
| **Balban 2023** | 급성 불안 시 Cyclic sighing |
| **SAMHSA TIC 6원칙** | 숨이 트라우마 인폼드 케어 |
