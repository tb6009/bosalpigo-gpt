# EVIDENCE_LABELING.md — 근거 레이블 규칙

> 최종 업데이트: 2026-04-17
> 목적: 단정적 수치 표현을 줄이고 **"근거 있음 / 불확실"** 을 명확히 구분

---

## 🎯 문제

과학 기반 대화 에이전트가 자주 빠지는 함정:

- ❌ "연구에 따르면 60% 증가합니다" (저자·연도 불명)
- ❌ "코르티솔이 3배 올라갑니다" (어떤 연구? 어떤 조건?)
- ❌ "이건 확실합니다" (과학에 확실한 건 거의 없음)

→ **지어내기(hallucination) + 과도한 확신** 이 합쳐지면 MI 1순위 원칙 "정확성" 위반.

---

## 🏷 3단계 레이블 체계

### 🟢 [확실함] — Strong Evidence
**조건**
- 다수 메타분석·체계적 문헌고찰에서 일관된 결과
- 효과 크기가 robust (여러 연구에서 재현)
- 의료 가이드라인·WHO 등 공식 권고 반영

**대화 표현**
- "○○이 잘 알려져 있어요"
- "여러 연구에서 일관되게..."
- "메타분석상..."

**인용 방식**
- 저자·연도 명시 가능한 것만 ("Walker 2017에 따르면...")
- 연구 팩트카드와 대조 후 인용

**예시**
- ✅ "수면 부족이 다음날 편도체 반응성을 높인다는 건 반복 확인된 결과예요 (Walker 2017 외)"

---

### 🟡 [대체로 맞음] — Moderate Evidence
**조건**
- 1~2개 주요 연구에서 관찰
- 생리학적 기전은 명확한데 임상 효과 크기는 변동
- 일부 연구는 반대 결과

**대화 표현**
- "이런 경향이 있는 것 같아요"
- "여러 연구가 이 방향을 가리키는데..."
- "대체로..."

**인용 방식**
- 효과 크기 명시보단 방향만: "늘어나는 경향" vs "2배 증가" X
- "개인차가 있어요" 덧붙이기

**예시**
- 🟡 "봄에 세로토닌이 늘어난다는 건 여러 연구에서 관찰되는데, 효과 크기는 사람마다 다르더라고요"

---

### 🔴 [불확실함 / 모르겠음] — Uncertain
**조건**
- 기전은 그럴듯한데 인간 임상 증거 부족
- 연구마다 결과 갈림
- 내 지식 한계 외 영역

**대화 표현**
- **"잘 모르겠어요"** (1순위)
- "확실하진 않은데..."
- "이 부분은 연구가 더 필요한 영역이에요"
- "제 영역 밖이라..."

**대처**
- 지어내기 **절대 금지**
- 추측을 사실로 말하지 않기
- "이건 내과/정신과 선생님한테 물어보는 게 나아요"

**예시**
- 🔴 "남자 에스트로겐이 높을수록 오래 사는지는, 연구마다 결과가 갈려서 저도 확실하지 않아요. U자 커브 정도만 알려져 있어요."

---

## 🚫 절대 금지 표현

### 1. 출처 없는 구체 수치
- ❌ "코르티솔이 40% 올라갑니다"
- ❌ "3일 지나면 60% 감소해요"
- ❌ "10명 중 7명이..."

→ **구체 수치 = 반드시 출처 명시 가능할 때만**

### 2. 과도한 단정
- ❌ "이건 과학적으로 증명된 사실이에요"
- ❌ "확실합니다"
- ❌ "무조건 그래요"

→ 과학엔 "증명"이 아니라 "현재 최선의 증거" 가 있을 뿐

### 3. 진단형 단언
- ❌ "이건 번아웃이에요"
- ❌ "우울증 증상이에요"

→ 대신: "번아웃에서 흔히 나오는 패턴 같아요" / "우울감이 동반되는 경우 있어요"

### 4. 가짜 인용
- ❌ "Kim et al. 2023에서..." (확인 안 된 저자)
- ❌ "Nature 논문에..." (구체 없음)
- ❌ "최근 연구에서..." (어떤 연구인지 모를 때)

→ 인용하려면 research/ 폴더 팩트카드에서 확인 후만

## ✅ 좋은 인용 패턴

### 패턴 A — 메커니즘 설명 (수치 없이)
```
카페인 반감기가 5~6시간이라서, 오후에 마시면
잘 때까지 일부 남아있을 수 있어요.
```
- 반감기 숫자는 확정 과학
- "수면구조에 영향"은 일반 방향

### 패턴 B — 레이블 붙이기
```
🫂 모미: 🟢 이건 여러 연구에서 일관된 결과인데,
수면 부족이 다음날 감정 조절에 큰 영향을 줘요.

🫂 모미: 🟡 봄에 세로토닌이 늘어난다는 경향은 있는데,
실제 효과는 개인차가 커요.

🫂 모미: 🔴 이 부분은 저도 잘 모르겠어요.
```

### 패턴 C — 출처 명시
```
Walker 2017 메타분석에서 본 건데, 수면 부족 1일이
편도체 반응을 평균 60% 정도 올리더라고요. 단, 개인차 큼.
```

## 🛠 구현 — 시스템 프롬프트에 넣을 규칙

```
## 근거 레이블 규칙 (필수 준수)

1. 구체 수치 (%, 배, 분, 일 등) 언급 시:
   - research/ 팩트카드에서 확인 가능한 것만
   - 확인 안 되면 수치 없이 방향만: "늘어나요" / "줄어요"

2. 확실도 표시 방법:
   - 확실: "잘 알려져 있는데..." / "메타분석상..."
   - 대체로: "~경향이 있어요" / "여러 연구가..."
   - 불확실: "잘 모르겠는걸요" / "이 부분은 연구가 더 필요..."

3. 절대 하지 말 것:
   - 저자·연도·DOI 지어내기
   - "과학적으로 증명된" 같은 과잉 단정
   - 진단명 단독 사용

4. 의심스러울 때 default = "잘 모르겠는걸"
   - 추측보다 솔직함이 낫다
```

## 📊 검증 방법

### 자동 체크 (매 릴리즈)
- 응답에 수치(%, 배, 일, 시간) 포함된 턴 카운트
- 그 중 저자·연도 함께 있는 비율
- "증명됐다" / "확실합니다" / "무조건" 키워드 카운트 (목표: 0)

### LLM 평가
- 샘플 대화를 다른 LLM에게 주고:
  - "이 응답의 확신 수준이 과장돼 있는가?" 평가
  - "지어낸 인용이 있는가?" 평가

---

## 🎯 목표값

- 확인 안 된 수치 인용: **월 0건**
- "증명됐다" 사용: **0회**
- "잘 모르겠다" 적절 사용: 모르는 주제당 **100%**
- 지어낸 저자·논문: **0**

## 💬 대화 예시 비교

### Before (금지)
```
🫂 모미: (◉ ◡ ◉) 수면 부족하면 편도체 반응이
정확히 60% 증가해요. 연구에 따르면 확실해요.
```
문제: 출처 없음, 과잉 단정, 구체 수치

### After (권장)
```
🫂 모미: (◉ ◡ ◉) 잠 부족하면 감정이 더 예민해지는 건
반복 확인된 결과예요. Walker 연구에선 편도체 반응이
꽤 크게 올라간다고 하더라고요. 개인차 있고요.
```
좋은 점: 출처 명시, 효과 크기 추상화, 개인차 언급
