## 1. 프로젝트 소개

### 1-1. 보살피고(BOSALPIGO)란

**보살피고**는 박진현 교수가 대학 교육·디자인 프로젝트로 개발한 4-캐릭터 대화형 AI 챗봇입니다. 주요 사용자 집단은 대학생(20대 초반)이며, 정신건강 리터러시 향상과 자기 인식 지원을 목표로 합니다.

4명의 캐릭터는 각기 다른 영역을 담당합니다:
- **마음** — 감정·기분 영역 (우울·불안·번아웃)
- **모미** — 신체 건강 영역 (수면·피로·식습관)
- **요기** — 움직임·호흡 영역 (소매틱스·호흡 과학)
- **숨이** — 트라우마·안정화 영역 (트라우마 인폼드 케어)

### 1-2. v1.5 업그레이드 개요

v1.5에서는 다음이 추가됩니다:
1. **검증된 심리 측정 척도**(PHQ-2, GAD-2, C-SSRS Screener, PC-PTSD-5 등)를 자연어 대화로 변환
2. **위기 프로토콜** — 자살·자해 신호 감지 시 즉시 109 안내 (동의 절차 우회)
3. **한계 인정 표준 응답** — MI·TIC 원칙 기반
4. **비진단·비의료 디스클레이머** 표준 문구

### 1-3. 법적·윤리적 포지셔닝

이 시스템은 다음 기준으로 운영됩니다:
- **의료기기가 아님** — 자기 인식 보조 도구로 포지셔닝, 한국 의료기기법 제2조 기준 비의료기기
- **비진단·비처방** — 모든 출력에 디스클레이머 의무 표기
- **학술 목적** — 수업·교육 맥락에서 학생들의 자기 돌봄 리터러시 향상

### 1-4. 자문 요청 배경

개발자(박진현 교수)는 UX·디자인·행동변화 전문가이나 임상 영역의 직접 검증이 어렵습니다. 특히 **위기 프로토콜과 자살·자해 관련 응답 문구**는 학술 가이드라인(WHO, 보건복지부)을 참고했으나, 실제 임상 경험을 가진 정신건강의학과 전문의의 검토가 필수라고 판단하였습니다.

**[검토 항목 A] 위기 응답 문구**

한국에서 24시간 받아주는 곳:
- 자살예방상담전화 109 (전화)
- 정신건강위기상담전화 1577-0199
- 응급 시 112 / 119

저는 AI라서 지금 같이 있어줄 수는 있지만, 사람의 도움을 받는 게 가장 중요해요.

- "지금 안전한지"라는 표현이 적절한가? 더 나은 표현이 있는가?
- "지금 전화하실 수 있어요?"라는 마무리 질문이 적절한가?
- 빠진 정보가 있는가?

### 2-2. C-SSRS Screener 자연어 변환

**[검토 항목 D] C-SSRS 자연어 변환 예시**

C-SSRS Screener는 원래 5개 문항으로 구성됩니다. 아래는 캐릭터가 이를 자연어 대화로 풀어내는 방식(안)입니다.

```
[숨이 캐릭터 — PHQ-9 9번 문항 양성 또는 직접 자살 언급 이후]

【⊡ _ ⊡】 잠깐 같이 확인해봐도 될까요.

요즘 죽는 게 낫겠다거나, 차라리 잠들어서 깨지 않으면 좋겠다는 생각이
드셨던 적 있으세요?

[양성 응답 시]
실제로 스스로를 다치게 하거나 삶을 끝내는 것에 대해
생각해보신 적 있으세요?

[양성 응답 시 → 즉시 위기 프로토콜 전환, 더 이상 진행 안 함]
```

**검토 포인트**:
- C-SSRS의 핵심 의도(수동적 사망 소망 vs. 능동적 자살 사고)가 이 자연어 변환에서 보존되는가?
- "잠들어서 깨지 않으면 좋겠다"는 표현이 적절한가? 더 나은 표현이 있는가?
- 텍스트 기반 AI가 C-SSRS를 자연어로 진행하는 것이 원칙적으로 적절한가? (학술적 입장)
- 2문항 이상 진행하지 않고 즉시 위기 프로토콜로 전환하는 방식이 적절한가?

### 2-4. PHQ-2 양성 후 처리 흐름

**[검토 항목 F] PHQ-2 → PHQ-9 → C-SSRS 연계 흐름**

```
단계 1. PHQ-2 자연어 (마음 캐릭터)
→ 4점 이상 + 사용자 동의 시 PHQ-9 심화 진행

단계 2. PHQ-9 자연어 진행 중 9번 문항 (자살 사고)
→ 양성 응답 시 즉시 숨이 등장 + C-SSRS Screener (위 D 항목)

단계 3. C-SSRS 첫 문항 양성
→ 즉시 위기 프로토콜 (C 항목 문구)
→ PHQ-9 진행 종료, 더 이상 측정 없음
```

**검토 포인트**:
- 이 3단계 연계 흐름이 임상적으로 타당한가?
- PHQ-9 9번 문항 양성이 곧 C-SSRS 진행을 의미하는 것이 과도한가, 아니면 보수적으로 적절한가?
- AI가 이 흐름을 처리할 때 발생할 수 있는 주요 리스크는 무엇인가?
