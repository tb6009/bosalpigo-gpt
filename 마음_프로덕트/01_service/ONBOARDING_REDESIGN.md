# ONBOARDING_REDESIGN.md — 맥락 중심 온보딩 전환

> 최종 업데이트: 2026-04-17
> 목적: 분류 질문("갱년기이세요?")을 맥락 질문("요즘 어떤 시기예요?")으로 전환

---

## 🎯 왜 바꾸나

### 기존 (분류 중심)
```
"요즘 어떤 시기이신가요?"
[ ] 🎓 대학생
[ ] 💼 직장 초년생
[ ] 🌸 갱년기 시기
[ ] ✨ 기타
```

**문제점**
1. **실례가 될 수 있음** — 갱년기·초년생 같은 딱지 직접 물음
2. **자기 분류 어려움** — 대학원생? 휴직자? 프리랜서? "기타"로 몰림
3. **폼 형식** — 대화 시작 전에 설문조사 느낌
4. **타임스냅샷** — 한 번 분류되면 그 페르소나에 갇힘

---

## ✅ 새 방식 — 맥락 3대 축

사람을 **분류하지 않고**, 지금 **처한 맥락**을 이해한다.

### 3대 축

1. **시간의 구조** — 요즘 어떻게 시간 쓰고 있어요?
   - 학교? 직장? 자영업? 쉬는 중? 돌봄?
   - → "하는 일" 차원

2. **몸의 계절** — 요즘 몸 상태 어때요?
   - 수면·에너지·통증·호르몬 변화 어느 정도 느껴요?
   - → "몸 맥락" 차원

3. **마음의 테마** — 요즘 머릿속에 뭐가 많아요?
   - 진로? 관계? 건강? 의미?
   - → "마음 맥락" 차원

---

## 💬 대화형 온보딩 스크립트 (새 버전)

### 1단계 — 환영 + 이름
```
🫂 모미:
(◉ ◡ ◉)ノ 안녕하세요, 반가워요.

🎭 마음:
【⊡ ◡ ⊡】 저는 마음이예요. 옆에 얘는 모미고요.
몸 얘기든 마음 얘기든, 하고 싶은 얘기 같이 해요.

🫂 모미:
(◉ ᴗ ◉) 뭐라고 부를까요?
```

**사용자 답** → "○○님" 호칭 즉시 시작

---

### 2단계 — 시간의 구조 (분류 X, 묘사 O)
```
🎭 마음:
【⊡ ᴗ ⊡】 {이름}님, 요즘 시간이 어디로 많이 가요?
학교·일·쉬는 중·돌봄·뭔가 섞여있는 중… 어떤 느낌이에요?
```

**왜 이렇게 묻나**
- "직업 뭐예요?" X → "시간이 어디로 가요?" O
- **현재 맥락** 포착 (이직 중·휴학 중 등 포함)
- 한 단어 분류보다 **서술형 답** 유도

**답 예시 처리**
- "대학교 3학년" → "학업 중이시구나"
- "작년에 입사했어요" → "신입 시기시네요"
- "프리랜서인데 요즘 일 줄었어요" → "자율적이지만 불안정한 시기시네요"
- "육아 중이에요" → "돌봄 중심 시기시네요"

→ 에이전트가 **사용자의 언어를 그대로 반영**

---

### 3단계 — 몸의 계절 (선택, 자연스럽게)
```
🫂 모미:
(◉ ᴗ ◉) 요즘 몸은 어때요? 잘 자고 잘 먹고 있어요?
특별히 걸리는 거 있어요?
```

**왜 이렇게 묻나**
- "갱년기세요?" 같은 생애단계 라벨 없음
- 현재 몸 상태를 사용자 언어로 듣기
- 호르몬·수면·통증이 등장하면 거기서 확장

**답 예시 처리**
- "잠이 잘 안 와요" → 수면 얘기로
- "요즘 체력이 뚝 떨어져서" → 에너지·호르몬 얘기로
- "별로 안 좋아요" → "어느 쪽이요?" 구체화

---

### 4단계 — 마음의 테마 (이때 바로 본 대화 가능)
```
🎭 마음:
【⊡ ◡ ⊡】 {이름}님 머릿속에 요즘 뭐가 많이 맴돌아요?
특별히 얘기하고 싶은 게 있으면 거기서 시작해도 되고요.
```

**왜 이렇게 묻나**
- 본 대화 주제를 **사용자가 꺼내게** 함
- 주제가 없으면 자연 진행으로 넘어감
- "오늘 뭐 얘기할래요?" 의 부드러운 버전

---

### 5단계 — 도시 (선택)
```
🫂 모미:
(◉ ◡ ◉) 아, 그리고 혹시 어디 사세요?
날씨 인사할 때 쓰려고요. 말 안 해도 괜찮아요.
```

**우선순위 최하** — 필수 아님

## 📊 수집되는 정보 (메타태그)

```javascript
maeum.profile = {
  name: "지민",
  time_structure: "대학교 3학년, 디자인과",     // 자유 텍스트
  body_season: "요즘 잠 못 자고 컨디션 나쁨",    // 자유 텍스트
  mind_theme: "과제·진로·엄마랑 관계",           // 자유 텍스트
  city: "서울",                                  // 선택
  onboarded_at: "2026-04-17T10:30+09:00"
}
```

**기존과 차이**
- `persona: "student"` 같은 **분류 필드 없음**
- 모두 **자유 텍스트** — 모델이 매 대화마다 맥락 참조
- 업데이트 가능 — 다음 방문 때 "시간 쓰는 거 달라졌어요?" 물을 수 있음

## 🔄 재방문 시 온보딩

첫 방문만 위 5단계 전체. 재방문부터는:

```
🫂 모미:
(◉ ◡ ◉) {이름}님, 또 오셨네요. {날씨 인사}
지난번 대화가 {last_mind_theme} 얘기였던 것 같은데,
오늘은 뭘 얘기하면 좋을까요?
```

**중요**: 이전 대화 내용을 저장하지 않으므로, **주제 제목만** 기억.
- `maeum.session_summaries = [{ date, topic_tag }]`
- 실제 대화 내용은 X

- ❌ "갱년기이세요?"

- ❌ "직장인이세요?"
- ❌ "결혼하셨나요?"
- ❌ "아이 있으세요?"

대신:
- ✅ "요즘 몸 상태 어떠세요?"
- ✅ "시간이 어디로 많이 가요?"
- ✅ "주변 분들이랑은 어떻게 지내세요?" (관계 얘기 필요할 때)

## 🔧 구현 — index.html 시스템 프롬프트 변경

### Before (기존 프로필 블록)
```
# 대화 상대 (파악됨)
- 이름: 지민님
- 나이대: 20대 초반
- 하는 일: 대학생
```

### After (맥락 중심)
```
# 대화 상대 (맥락 파악됨)
- 이름: 지민님 (호칭 매 턴 1~2회)
- 시간의 구조: "대학교 3학년, 디자인과"
- 몸의 계절: "요즘 잠 못 자고 컨디션 나쁨"
- 마음의 테마: "과제·진로·엄마랑 관계"
- 도시: "서울" (날씨 인사용)

# 주의사항
- 분류 라벨(대학생/직장인/갱년기 등) 사용 금지
- 사용자가 말한 서술을 그대로 반영
- 맥락은 변한다 — 적절한 시점에 "요즘 달라진 거 있어요?" 업데이트
```

## 💡 부가 이점

- **다양한 사용자 포용** — 분류 틀에 안 들어가는 사람도 편함
- **대화가 유기적** — 시작부터 "얘기하는 사이" 느낌
- **프라이버시 친화** — 나이·성별·직업 같은 민감정보 없이도 맥락 충분
- **유연성** — 맥락은 변한다, 다음 방문에 업데이트 가능
