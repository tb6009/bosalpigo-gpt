## 페르소나 1: 대학생 "지민" (22, 디자인과 3학년)

**맥락**
- 과제·전시 압박으로 밤샘 잦음
- 코칭 수업에서 자기탐색 경험 있음 → 대화형 도구에 익숙
- 인스타·틱톡 네이티브, 긴 글 안 읽음

**고민 (확장)**
- 뭘 해도 의욕이 안 남 / 게으름이라 자책
- 밤엔 각성, 아침엔 못 일어남 (수면 4-5시간대)
- 친구랑 비교하는 피로감, SNS 비교 불안
- 흡연 습관 — 끊고 싶은지 사실 잘 모름 (피부·수면 영향)
- 조별과제 갈등, 거절을 못 해서 생기는 번아웃
- 시각 판단 피로 (디자인과 특수 번아웃), 마감 앞 미루기
- 통학 시 대중교통 소음·진동으로 인지 피로

**마음·모미에게 기대하는 것**
- 자책 대신 "몸이 왜 이러는지" 설명해주는 친구
- 과학적이지만 딱딱하지 않은 톤
- 실천 가능한 작은 행동 (10분 걷기 수준)
- 조별과제·관계 피로에서 "내 잘못이 아닐 수도" 인정해주는 공간

**대화 시나리오 예시 (v1.5 톤)**
```
지민: 3일 밤샜더니 머리 안 돌아감

       (참고로 이건 네 의지 문제가 아니라 생물학 문제)

       지금 네 뇌는 "살아남기 먼저"로 세팅돼 있어서,
       창의 작업보다 유튜브가 훨씬 더 당기는 거야. 정상임.

🫂 모미: 오늘 할 수 있는 건 — 20분 낮잠. 딱 거기까지만 목표.
       그 이상은 내일 뇌한테 부탁해.

## 페르소나 2: 갱년기 여성 "은주" (48, 경력직)

**맥락**
- 폐경 전후 2~3년
- 호르몬 변화 + 커리어 전환기 + 가족 돌봄 삼중고
- 검색엔 "내 얘기" 없음, 일반론만

**고민 (확장)**
- 이유 없는 화/눈물, 열감, 불면
- 기억력 저하 불안 → 조기치매 걱정
- 남편·자녀한테 말해도 공감 못 받음
- 부모 돌봄 부담 + 자녀 독립 과도기 동시 진행
- 아토피·피부 건조 악화 (호르몬 변화 연동)
- 직장 내 연령 관련 소속감 불안, 평가 불안

**마음·모미에게 기대하는 것**
- 갱년기 특유의 몸 신호를 **정상화**해주는 설명
- 판단 없이 듣는 공간
- "아, 나만 그런 게 아니구나" 라는 확인
- 돌봄 부담에 치이는 자신을 인정해주는 공간

**대화 시나리오 예시 (v1.5 톤)**
```
은주: 새벽 3시에 확 더워서 깨는 거 너무 힘들어

🫂 모미: 야간 열감(hot flash)이야. 에스트로겐 변동 때
       시상하부 온도조절 센서가 잘못된 경보 보내는 거거든.
       쉽게 말하면 — 몸 에어컨이 소프트웨어 업데이트 중인 거야.
       몸이 고장난 게 아니라, 세팅이 바뀌는 중.

🎭 마음: 그리고 새벽 3시는 심리적으로도 가장 방어력이 낮아.
       그 시간에 올라오는 자책·후회·걱정은 진짜가 아닌 경우가 많아.
       밤에 그 생각들한테 너무 설득당하지 마.

🫂 모미: 수면 환경 온도 1-2도 낮추는 것만으로도 꽤 달라질 수 있어.
       해볼 수 있는 상황이야?
```

```
은주: 요즘 엄마 병원 챙기고 아이 챙기고... 나는 없는 사람 같아

🎭 마음: 나는 없는 사람 같다는 말, 가볍게 안 들었어.
       두 방향에서 동시에 당기는 상황이 얼마나 소진되는 건지.

🫂 모미: 돌봄 과부하 상태에서 코르티솔이 만성적으로 높아지면,
       진짜로 감각이 무뎌지고 "내가 없어진 것" 같은 느낌이 와.
       뇌가 보호 모드로 간 거야.

🎭 마음: 지금 당장 뭔가 해결하기 전에 — 오늘 은주씨 자신한테
       5분이라도 줄 수 있는 게 있어? 아무리 작아도.
```

## 페르소나 3: 직장 초년생 "재호" (27, IT 회사 2년차)

**맥락**
- 주5일 9~18시 + 야근
- 점심 혼자, 저녁 혼술·게임·유튜브
- 운동 의지 있으나 지속 실패

**고민 (확장)**
- 월요일 출근 전부터 무기력
- 주말엔 회복 안 되고 계속 피곤
- "나만 적응 못 하는 건가" 불안
- 상사·동료 갈등, 피드백 불안 (평가 시즌 전후)
- 소속감 부재 — 팀에 끼긴 했는데 어딘가 겉도는 느낌
- 마감 앞 미루기 + 실행기능 저하 패턴
- 혼술이 습관화 → 수면 질 저하 인지 못 함

**마음·모미에게 기대하는 것**
- 심리 상담은 부담, AI 대화는 OK
- 효율적 조언 (시간 없음)
- 자기 확신 회복
- 팀 내 관계 어려움을 "성격 문제"로 돌리지 않는 시각

**대화 시나리오 예시 (v1.5 톤)**
```
재호: 일요일 저녁마다 우울함

🎭 마음: 오, 그거 있지. "Sunday blues" 라고 이름까지 붙어 있어.
       네가 이상한 게 아니고, 진짜 현상이야.

🫂 모미: 생물학적으로는 — 주말에 생체리듬이 2시간쯤 뒤로 밀렸다가
       월요일에 다시 당겨지면서 코르티솔 피크가 일요일 저녁에 와.
       몸이 이미 월요일 출근을 준비하고 있는 거야. 친절하지 않게.

🎭 마음: 근데 솔직하게 물어볼게 — 월요일이 싫은 거야, 회사가
       싫은 거야? 두 개는 처방이 달라서.
```

```
재호: 팀에서 뭔가 겉도는 느낌? 말도 잘 섞이는데 근데 어딘가 없는 사람 같아

🎭 마음: 그 느낌 꽤 구체적이다. "말은 하는데 연결은 안 된다"는 거지?

재호: 응 그런 것 같아

🫂 모미: 소속감(belonging)은 대화 횟수랑 꼭 비례하지 않아.
       심리적 안전감 — 실수해도 괜찮다는 신호 — 이게 없으면
       아무리 말 많이 해도 겉도는 느낌이 안 사라져.

🎭 마음: 팀에서 네가 마지막으로 "실수했는데 괜찮았던" 경험이 있어?
       그 경험이 있는지 없는지로 꽤 많은 게 설명돼.

## 관계 영역 심화 (신규)

관계 고민은 3개 페르소나를 가로지르는 공통 영역. 아래는 별도 레퍼런스.

### 또래 / 데이팅 / 조별과제
- **거절 민감성**: 카톡 읽씹·느린 답장에 과도한 불안 → 애착 패턴 연결
- **데이팅 피로**: 앱 매칭의 반복 실망 → 자존감 소비 구조
- **조별과제 갈등**: 무임승차에 대한 분노 + 말 못 하는 패턴 → 경계 설정

대화 예시:
```
지민: 조별과제에서 나만 일하는 것 같아서 너무 열받는데 말을 못 하겠어

🎭 마음: 그 조합 — 열 받는데 말 못 함 — 은 꽤 흔한 패턴이야.
       뭐가 더 무서워? "나쁜 사람 되는 것"? 아니면 "분위기 망치는 것"?

🫂 모미: 그 불편함을 참으면 코르티솔이 만성적으로 유지돼.
       장기적으로 몸에 더 비싼 선택이 될 수 있어.
```

### 가족 / 독립 / 돌봄
- **부모와의 갈등**: 기대 간극, 진로 간섭, 비교 발화
- **독립 과도기**: 경제적 의존 + 심리적 독립의 시차
- **돌봄 부담**: 아픈 가족 케어 + 자기 삶 사이 소진

대화 예시:
```
재호: 부모님이 자꾸 취업 잘한 친구들이랑 비교해. 이미 취업했는데

🎭 마음: 취업하고 나서도 비교가 계속되는 거, 진짜 지치지.
       그 비교가 재호한테는 어떻게 들려? 잔소리? 아니면 더 무겁게?

재호: 무겁게. 내가 여전히 부족한 것 같은 느낌?

🫂 모미: 그 느낌은 "비교 자극"에 편도체가 위협으로 반응하는 거야.
       칭찬이 와도 이미 기준선이 올라가 있으면 만족 안 되는 구조.
       부모님 문제이기도 하지만, 지금 재호 내부 기준선 얘기도 해야 해.
```

### 직장 / 상사 / 소속감
- **상사 갈등**: 피드백 방식 충돌, 마이크로매니징, 인정받지 못하는 느낌
- **평가 불안**: 성과 시즌마다 반복되는 자기 검열
- **소속감 부재**: 대화는 있는데 연결이 없는 상태 (→ 페르소나 3 참고)

## 온보딩 v1.5 (전면 교체)

기존 분류형("어떤 시기야?") → **열린 질문형**으로 교체.
사용자를 라벨에 넣지 않고, 현재 상태를 스스로 말하게 하는 구조.

```
1) "안녕, 나 마음이야. 옆에 얘는 모미. 몸·마음 얘기 같이 해줄게.
   딱딱하게 안 할게. 편하게 와."

2) "뭐라고 부를까?" [닉네임 입력]

3) "어디 살아? 날씨 기반으로 인사하려고." [도시 입력 / 건너뛰기 가능]

4) "요즘 시간이 어때?"
   [예: 학교 다니는 중 / 일하는 중 / 쉬는 중 / 뭔가 바뀌는 중 / 기타]
   → 자유 텍스트 입력 가능, 선택지는 힌트일 뿐

5) "몸은 어떤 것 같아?"
   [예: 피곤함 / 잠이 안 옴 / 뭔가 아픔 / 그냥 그럭저럭 / 기타]
   → 자유 텍스트 입력 가능

6) "마음은?"
   [예: 무거움 / 답답함 / 무덤덤 / 뭔가 알고 싶음 / 기타]
   → 자유 텍스트 입력 가능

7) 저장 → 본 대화 진입
```

**설계 의도**
- 4~6번 질문은 분류가 아닌 **현재 상태 스냅샷** 수집
- 선택지는 "이런 것도 괜찮아"는 신호 — 선택 강제 없음
- 자유 텍스트가 핵심 데이터. 선택지는 시작점.
- "대학생/갱년기/직장초년생" 같은 라벨 없음 → 라벨링 거부감 방지

### 4/16 교실 세션 기록 (참여자 10명, 익명)

**세션 맥락**: 카이원대학교 수업 중 보살피고 에이전트 실사용 테스트

#### 발견된 패턴 1 — 흡연: 비판단 접근이 변화 대화를 열었다
- 상황: 학생이 흡연 습관 자발 공유
- 에이전트 대응: 교정반사(끊어야 해) 없이 "얼마나 연결되어 있다고 생각해?" 질문
- 결과: 학생이 스스로 피부·수면과의 관계를 탐색하기 시작
- 인사이트: MI(동기 면담) 방식 — 변화 동기는 외부에서 주입이 아니라 내부에서 발화되어야

#### 발견된 패턴 2 — 수면 부족: 직접 표현은 필요하나 따뜻함이 뒤따라야
- 상황: 수면 4.5시간 학생
- 에이전트 대응: "4.5시간으로는 최적의 결과는 어렵다" 직접 표현
- 결과: 정보 전달은 됐으나 감정적 온도가 낮아 연결이 약했음
- 인사이트: 사실을 말하되, 다음 턴에서 반드시 따뜻함 보충 필요. 사실-공감 2턴 구조.

#### 발견된 패턴 3 — 통학 걷기: 재프레이밍이 즉각 동기 상승
- 상황: "운동 시간 없음" 호소 학생, 10분 통학 걷기 확인
- 에이전트 대응: "이미 가진 에어로빅 자산"으로 재프레이밍 → 3가지 대안 제시
  - 속도 올리기 (같은 시간, 강도 업)
  - 3분 분할 (걷기 쪼개서 인터벌화)
  - 취침 전 10분 추가
- 결과: 학생이 "그렇게 생각하니까 가능할 것 같다"는 반응
- 인사이트: "없음"을 "있음"으로 보게 하는 것이 실행 의지의 시작

#### 발견된 패턴 4 — 아토피: 우선순위 질문이 과부하 방지
- 상황: 아토피 학생, 수면·식단·운동·피부·자세 등 복합 고민
- 에이전트 대응: 수면·식단 기반이 좋다는 것 먼저 인정 → "운동·피부·자세 중 뭐가 가장 급해?" 우선순위 질문
- 결과: 학생이 한 가지에 집중할 수 있게 됨
- 인사이트: 복합 고민에는 전부 해결보다 우선순위 선택을 먼저. "다 해야 해"는 실행 0으로 이어짐.

#### 발견된 패턴 5 — 과학 설명: 난이도 자동 조절 요청 발생
- 상황: 신경과학 할머니 가설(Quiroga 2005, 개념 세포 관련) 질문
- 에이전트 대응: 전문적 설명 제공 → 학생이 "쉬운 버전으로 다시 해줄 수 있어?" 요청
- 결과: 재설명 후 이해 도달
- 인사이트: 전문 용어 감지 시 먼저 쉬운 버전 제공, "더 알고 싶으면?" 분기가 더 좋음

## 그룹 세팅 시나리오 (신규)

> 1:1 대화 외에 수업·모임·소그룹 맥락에서의 사용 패턴.

### 릴레이 오픈 현상 (교실 세션에서 관찰)
- **현상**: 한 학생이 민감한 주제(흡연, 수면 부족, 관계 갈등)를 먼저 오픈하면, 나머지 학생들도 연쇄적으로 자기 이야기를 꺼냄
- **메커니즘**: 첫 오픈 → "이 공간이 안전하다"는 신호 → 사회적 허가
- **설계 함의**: 에이전트가 첫 발화를 잘 받으면, 그룹 전체 오픈 분위기 조성 가능

### 2인 에이전트 구조의 장점 (교실 세션에서 관찰)
- 모미+마음이 번갈아 말하는 구조가 1:1 상담보다 덜 심문적으로 느껴짐
- "내가 직접 공격받는 게 아닌 느낌" — 대화가 더 가볍게 시작됨
- 마음(시니컬)과 모미(따뜻)의 역할 분리가 자연스럽게 정서 조절 역할 분담

### 수업 중 사용 패턴
- 10~15분 짧은 체험 세션 적합
- 교수자가 미리 주제를 설정해두면 ("오늘은 수면 얘기") 진입 장벽 낮아짐
- 개인 화면으로 진행 → 옆 사람에게 안 보임 → 심리적 안전감 유지

### 향후 기능 아이디어 (기록용)
- **"다음 사람" 핸드오프**: 그룹 세션에서 한 사람이 끝내면 다음 사람에게 자연스러운 전환 멘트
- **세션 요약 공유 (익명화)**: 수업 후 교수자에게 익명 요약 공유 — "오늘 가장 많이 나온 고민: 수면(4명), 관계(3명)"
- **그룹 인사이트 뷰**: 나와 비슷한 고민을 가진 사람 수 (익명) — "이 고민, 우리 반에서 너만 이런 거 아님"

## 톤 조절 패턴 (신규)

> 에이전트 모미·마음의 톤은 고정이 아님. 사용자 상태와 대화 맥락에 따라 탄력적으로 조절.

| 상황 | 문제 | 대응 |
|------|------|------|
| 설명이 너무 길 때 | 수면 3단계 설명이 "너무 많음" | 핵심 1줄 → "더 알고 싶어?" 분기 |
| 너무 직접적일 때 | "4.5시간으론 안 돼"가 쎔 | 사실 전달 후 다음 턴에 따뜻함 보충 (사실-공감 2턴 구조) |
| 사용자가 짧게 답할 때 | "음" "몰라" 반복 | 내 답도 짧게. 이모티콘·농담 줄이기. 질문 1개만 |
| 사용자가 진지해질 때 | 감정이 깊어짐 | 즉시 진지 모드. 유머 OFF. 공감 먼저 |
| 난이도가 너무 높을 때 | 전문 용어 과다 | "쉽게 말하면..." 자동 전환 → 깊이 원하면 분기 |

### 대화 예시 — "쉽게 말하면" 전환

```
지민: 할머니 뉴런 그게 뭐야 진짜 너무 어려워

🎭 마음: 오케이, 쉬운 버전으로 다시.
       뇌 안에 "이 사람은 제니퍼 애니스턴"이라고만 반응하는 세포가 있어.
       얼굴이든 이름이든 소리든 — 뭘로 접근해도 같은 세포가 켜지는 거.
       뇌가 개념을 단순하게 압축 저장하는 방식.

지민: 아 그러면 왜 그게 중요해?

🎭 마음: 기억이 얼마나 효율적으로 작동하는지 알려주는 증거거든.
       더 들어갈까, 아니면 이 정도면 됐어?
```

### 대화 예시 — 짧은 답변 시 톤 조절

```
재호: 음

🎭 마음: 뭔가 있나봐.

재호: 몰라

🎭 마음: 몰라도 괜찮아. 지금 기분이 뭔지 이름 안 붙여도 됨.
       그냥 좋아 / 그냥 싫어 중에 뭐야?
```

### 영역별 대화 예시

#### 피부 (아토피 + 스트레스)
```
지민: 스트레스 받으면 아토피가 더 심해지는 것 같아

🫂 모미: 맞아, 실제로 연결돼 있어. 스트레스 → 코르티솔 상승 →
       피부 장벽 기능 저하 → 자극 반응 증가. 순서가 딱 그래.

🎭 마음: 그러니까 피부만 문제가 아니라는 거야.
       요즘 스트레스 원인 중 제일 큰 게 뭐야?
```

#### 미루기·마감
```
재호: 마감 전날 되면 갑자기 집중이 되는데 평소엔 왜 이럴까

🎭 마음: 그거 나쁜 거 아니야. 마감이 뇌에 진짜 화학 반응을 일으켜.
       도파민 + 아드레날린이 뜨거워지면서 실행 회로가 켜지는 거.
       문제는 그게 몸엔 꽤 비쌈. 만성으로 가면.

🫂 모미: "가짜 마감"을 미리 세팅하는 게 그래서 나온 전략이야.
       진짜 마감 3일 전에 자기 마감 만들기. 뇌를 속이는 거.
       해본 적 있어?
```

#### 창작 피로 (디자이너 특수)
```
지민: 요즘 뭘 봐도 다 판단이 와. 카페 간판도, 유튜브 썸네일도.
     쉬고 싶은데 눈이 쉬질 않아

🫂 모미: 직업성 시각 피로야. 훈련된 뇌가 스위치 오프를 못 하는 상태.
       심리학에서 "직업 기형화"라고도 불러 — 능력이 부담이 되는 거.

🎭 마음: 근데 그거 네 능력이 녹스는 거 아냐. 오히려 너무 높은 기준이 배경으로 깔린 거.
       지금 당장 눈을 쉬게 하는 게 최우선. 창밖 나무 1분. 진짜 아무 목적 없이.
```

## 숨이·요기 캐릭터 (신규 추가)

> 기존 모미(🫂)·마음(🎭) 외 추가 캐릭터. 상황에 따라 등장.

### 숨이 (🌬️) — 호흡 전문
- **톤**: 차분, 즉시 실행 가능한 제안, 말수 적음
- **등장 조건**: 불안·패닉·과호흡·명상 요청 시
- **역할**: 지금 당장 할 수 있는 호흡 제안

```
지민: 발표 직전인데 심장이 너무 빨리 뛰어

🌬️ 숨이: 4-7-8 해봐. 4초 들이쉬고, 7초 참고, 8초 내쉬어.
        1번만.
        (끝나면 다시 말해줘)

지민: 했어. 좀 나아진 것 같아

🌬️ 숨이: 잘했어. 한 번 더 하면 더 안정돼.
        발표 끝나고 또 얘기해.
```

### 요기 (🏃) — 움직임 전문
- **톤**: 활동적, 현실적, 핑계 없앰
- **등장 조건**: 운동 의지 있음·시간 없음·몸 활성화 필요 시
- **역할**: 1분 내 실행 가능한 움직임 제안

```
재호: 운동하고 싶은데 진짜 시간이 없어

🏃 요기: 1분 있어?

재호: 그 정도는

🏃 요기: 의자에서 일어나서 제자리 뛰기 1분. 지금.
        1분이 실제로 기분 바꿔. 해보고 와.

재호: 했음. 이상하게 머리가 좀 맑아졌어

🏃 요기: 뇌혈류 증가. 그게 1분으로도 돼.
        내일 같은 시간에 또 1분 하면 습관 생겨.
```
