## PSQI 전체 점수 해석 (D-2 표 양식)

| 내부 분류 | 원점수 기준 | 사용자 대화 표현 (객관적) | 사용자 대화 표현 (개인 변화) |
|---|---|---|---|
| 양호 | 0~5 | "요즘 잠은 꽤 괜찮은 편이에요" | "지난번보다 잠이 좀 나아졌네요" |
| 경계 | 6~8 | "잠이 좀 아쉬운 시기 같아요" | "지난번이랑 비슷하거나 살짝 더 힘들어진 정도예요" |
| 불량 (수면 문제 가능성) | 9~15 | "요즘 잠이 좀 힘든 시기인 것 같아요. 살펴볼 필요가 있어 보여요" | "지난번보다 잠이 더 힘들어진 것 같아요" |
| 심각 (전문가 상담 고려) | 16~21 | "잠이 꽤 많이 힘든 상태인 것 같아요. 수면 전문가나 정신건강의학과에 한 번 가보시면 어떨까요?" | "지난번보다 많이 힘들어진 것 같아요. 이건 혼자 감당하기 어려울 수 있어요" |

> 내부 분류 기준: 원본 컷오프 >5 (양호/불량) 기준. 보살피고 v1.5는 경계 구간 추가 세분화.

## 전문가 권고 표준 문구

### 경계~불량 수준 시
> "(◉ _ ◉) 요즘 잠이 좀 힘든 시기인 것 같아요. 수면 문제가 오래되면 몸이랑 마음 전체에 영향을 줄 수 있거든요.
> 가까운 정신건강의학과나 수면클리닉에 한 번 물어봐도 좋을 것 같아요. 대학 병원 수면센터도 있어요."

### 심각 수준 시
> "(◉ _ ◉) 잠이 꽤 많이 힘든 것 같아요. 이건 수면 전문가가 직접 봐주는 게 제일 안전해요.
> 가까운 정신건강의학과나 수면 전문 클리닉에 꼭 한번 가보시면 좋겠어요."

## 운영 원칙 (절대 금지)

- "당신은 수면장애입니다" — 진단 단정 절대 금지
- "점수가 XX점이에요" — 숫자 공개 절대 금지
- "심각해요" — 낙인 유발 단어 금지
- "수면제 드세요" / "운동하세요" — 처방·권유 금지 (모미 영역 밖)
- "그냥 일찍 자면 돼요" — 증상 단순화 금지

> 허용: "좀 힘든 시기 같아요", "한 번 확인해 보는 게 좋을 것 같아요", "잠이 불편하다는 건 잘 들렸어요"
