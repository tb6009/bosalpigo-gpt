# PC-PTSD-5 / PCL-5 재측정 주기

> 담당 캐릭터: 숨이 | 영역: 트라우마 | 갱신: 2026-04-25
> ⚠️ 트라우마 영역은 가장 보수적 주기 적용

---

## 학술 권장 주기

| 척도 | 권장 주기 | 출처 |
|---|---|---|
| PC-PTSD-5 (스크리닝) | 1개월 | National Center for PTSD 권고 |
| PCL-5 (심화 후 추적) | 1개월 | Weathers et al. (2013); VA/DoD 가이드라인 |
| PCL-5 (치료 중 모니터링) | 2~4주 | VA/DoD Clinical Practice Guidelines |
| PCL-5 (연구 목적) | 필요 시 주 1회 가능 | 단, 챗봇 맥락에서는 과도한 측정 자제 |

**근거**:
- PCL-5는 기본적으로 "지난 1개월" 기간 기준으로 개발됨 — 재측정 시 1개월 간격이 원칙.
- National Center for PTSD: PCL-5 월 1회 사용이 임상 표준.
- VA/DoD Clinical Practice Guidelines: 치료 반응 모니터링 시 2~4주 단축 가능하나, 임상 감독 하에서만.

**중요**: 트라우마 척도의 과도한 반복 측정은 재트라우마화(retraumatization) 위험이 있음.
보살피고 v1.5에서는 **최소 1개월 간격**을 기본값으로 설정.

## 보살피고 v1.5 적용 주기 (보수적 적용)

| 상황 | 적용 주기 | 비고 |
|---|---|---|
| PC-PTSD-5 정기 스크리닝 | **최소 1개월** | 기본값 |
| PCL-5 심화 후 추적 | **1개월** | 단축 불가 (보수성 원칙) |
| 사용자 자발 요청 시 | 최소 2주 간격 | 절대 최소 간격 |
| 위기 발생 시 | 즉시 C-SSRS + 위기 프로토콜 | 주기 계산 무관 |

## 자연 회상 표준 문구 (1개월 단위)

### 1개월 후 기본 회상
> "【⊡ ᴗ ⊡】 한 달 전에 이야기 나눴던 것, 요즘은 어때요? 같은 정도인가요, 아니면 달라진 게 있나요?"

### 양성이었던 경우 회상
> "【⊡ _ ⊡】 저번에 힘들었던 것들, 지금도 그런 영향이 있어요? 조금이라도 달라진 게 있으면 좋겠는데."

### 비양성이었던 경우 회상
> "【⊡ ᴗ ⊡】 저번에 비교적 안정적이었는데, 지금도 비슷한가요?"

## 재측정 트리거 조건

```
기본 트리거: 마지막 PC-PTSD-5 측정일 + 30일 경과
자동 회상 문구: 위 표준 문구 적용

비트리거 조건 (자동 제안 안 함):
- 사용자가 급성 스트레스 상황 언급 시 → 즉시 상황 대응 (측정 보류)
- 위기 신호 감지 시 → 즉시 위기 프로토콜 (주기 무시)
- 사용자가 "지금은 됐어요" 패턴 반복 → 자발적 진입까지 제안 중단
```

## 임상적으로 의미 있는 변화 기준

- **PCL-5**: 10점 이상 변화가 임상적으로 유의미한 변화
  - 개선 10점 이상: "많이 편안해진 것 같아요"
  - 악화 10점 이상: "더 힘들어진 것 같아요 — 전문가와 이야기해봐요"
  - 5점 미만 변화: "지난번이랑 비슷한 상태예요"

출처: VA PCL-5 사용 가이드 (https://www.ptsd.va.gov/professional/assessment/documents/using-PCL5.pdf)
