(알코올 의존 환자 대상 한국판 C-SSRS 신뢰도·타당도 연구)
*ResearchGate 확인 논문 — 정확 서지 추가 확인 필요*

- **ResearchGate**: https://www.researchgate.net/publication/282620168_The_Reliability_and_Validity_of_the_Korean_Version_of_Columbia-Suicide_Severity_Rating_Scale_in_Alcohol_Dependent_Patients
- **신뢰등급**: 한국 임상 검증 연구 (알코올 의존 환자 표본)
- **상태**: 부분 확인 — 저자·저널·연도 추가 확인 필요

#### 핵심 3줄 (한국어)
1. 알코올 의존 환자 대상 한국판 C-SSRS(K-C-SSRS)의 신뢰도·타당도 초기 검증.
2. 내적 일관성 일부 하위 척도에서 높음 — Beck Hopelessness Scale, BDI, SSI와 타당도 확인.
3. 한국 임상 환경에서 K-C-SSRS 사용 가능성 확인.

## 공식 한국어판 C-SSRS 현황

### Columbia Lighthouse Project 공식 한국어판
- **공식 한국어 번역**: Columbia Lighthouse Project에서 공식 한국어 번역본 제공
- **다운로드**: https://cssrs.columbia.edu (등록 후 무료 다운로드)
- **버전**: C-SSRS Screen Version (Recent) 한국어판 존재 확인
  - 참고: https://www.bluecloudx.net/wp-content/uploads/2025/04/M-Appendix-M-0108-STANDARDS-KOREAN-C-SSRS_v7.5.pdf

#### 핵심 확인사항
1. Columbia Lighthouse Project는 공식 번역본을 다국어로 제공 (한국어 포함).
2. 공식 번역본 사용을 권장 — 비공식 번안은 사용 금지.
3. 번역본 사용 시 Columbia Lighthouse Project 승인 절차 확인 필요.
