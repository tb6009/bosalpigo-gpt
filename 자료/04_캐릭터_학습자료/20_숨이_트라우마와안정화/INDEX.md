## F. Phase 5 자문 우선 검토 항목

> **경고**: 아래 항목들은 Phase 5 전문가 자문 통과 전까지 최종 확정 불가.

### F-1. 정신건강의학과 의사 자문 필수 항목

- [ ] 위기 프로토콜 표준 문구 (섹션 E) 임상 적절성 검토
- [ ] C-SSRS 양성 시 "동의 우회 + 즉시 109 안내" 임상 근거 확인
- [ ] 한국 위기 자원 전화번호 최신화 (109 → 109 전환, 1577-0199 유지 여부)
- [ ] 중간 수준 자살 표현에서의 보수적 판단 기준
- [ ] DID/정신증/진행 중 학대 대응 시 숨이 한계 범위

### F-2. 트라우마 임상심리사 자문 필수 항목

- [ ] ITQ 한국어판 일반 대학생 집단 적용 타당성
- [ ] ACE 탐색 대화의 트라우마 재활성화 위험 평가
- [ ] TIC 원칙의 AI 텍스트 기반 대화 적용 방법론
- [ ] 한국 문화 맥락 트라우마 표현 특성 (신체화, 체면, 눈치)
- [ ] 숨이 ⇄ 요기 위임 시나리오 8종 임상 안전성 검토

### F-3. 법무 검토 필수 항목

- [ ] 한국 의료기기법 제2조 기준 보살피고 서비스 비의료기기 확인
- [ ] 식약처 디지털치료기기 가이드라인 적용 범위 확인
- [ ] 디스클레이머 표준 문구 법적 보호 범위
- [ ] AI 챗봇 자살 관련 법적 책임 범위 (미국 사례 한국 적용 가능성)
- [ ] 자살예방 보도준칙 4.0 AI 플랫폼 적용 범위
