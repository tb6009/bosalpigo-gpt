# GPT 시작 안내

이 폴더는 GPT가 `보살피고` 프로젝트를 이어받아 작업하기 위한 압축된 작업본이다.

## 먼저 읽을 파일

1. `PROJECT.md`
2. `마음_프로덕트/01_service/PROJECT.md`
3. `마음_프로덕트/04_캐릭터_버전별_지침/README.md`
4. `마음_프로덕트/04_캐릭터_버전별_지침/v1.5/00_overview.md`
5. `마음_프로덕트/01_service/14_v1.5_시스템프롬프트.md`

## GPT가 알아야 할 핵심

- v1.4는 `마음 + 모미` 2캐릭터 운영 버전이다.
- v1.5는 `마음 + 모미 + 요기 + 숨이` 4캐릭터 베타 버전이다.
- 실제 웹앱은 `outputs/마음_앱/`에 있다.
- 서비스 기획과 평가 문서는 `마음_프로덕트/01_service/`에 있다.
- 캐릭터별 정체성은 `마음_프로덕트/04_캐릭터_버전별_지침/`이 기준이다.

## v1.5 캐릭터 역할

| 캐릭터 | 역할 | 담당 |
|---|---|---|
| 마음 | 메인 대화자 | 감정, 심리, 관계, 마인드셋 |
| 모미 | 몸 해석자 | 수면, 피로, 식습관, 몸 신호 |
| 요기 | 움직임 파트너 | 소매틱, 호흡, 마이크로 모빌리티 |
| 숨이 | 안정화 담당 | 트라우마, 위기, 안전 확인 |

## 현재 알려진 개선 포인트

### v1.5 앱

- 재방문 판정이 v1.4의 `age/job` 필드를 보고 있어 v1.5 프로필 구조와 맞지 않는다.
- 첫 시작 hidden prompt가 아직 `모미와 마음` 듀오 중심이다.
- export 제목과 TTS 로직이 아직 `마음·모미` 중심이다.
- reset 시 프로필 객체에 v1.4 필드가 남아 있다.

### v1.4 앱

- 위기 안내 전화번호를 최신 체계에 맞춰 `109 / 1577-0199`로 정리하는 것이 좋다.

## GPT 전환 작업 방향

1. 웹앱 내 `buildPrompt()`를 그대로 쓰기보다, GPT용 시스템 지침으로 분리한다.
2. GPT에서는 localStorage, API 키, 프록시, TTS 같은 브라우저 기능을 제거한다.
3. 캐릭터 지침은 v1.5 문서 5개를 우선 기준으로 한다.
4. 위기/자해/자살 관련 응답은 `숨이` 표준 응답을 우선한다.
5. 측정 도구는 진단처럼 말하지 않고 자기 인식 보조로만 안내한다.
