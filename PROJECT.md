# 10-일상다반사GPT

> 원본 `10-일상다반사`에서 보살피고/마음 프로젝트의 핵심만 복사한 GPT 작업용 폴더.
> 목적: GPT 환경에서 같은 서비스 프로젝트를 이해하고, 테스트하고, v1.5 베타를 개선할 수 있도록 최소 실행 패키지를 제공한다.

## 포함한 핵심 범위

```text
10-일상다반사GPT/
├── PROJECT.md
├── PROJECT_ORIGINAL.md
├── GPT_START.md
├── 마음_프로덕트/
│   ├── 01_service/
│   ├── 04_캐릭터_버전별_지침/
│   └── 12_모미마음이_Test/
├── outputs/
│   └── 마음_앱/
└── 자료/
    └── 04_캐릭터_학습자료/
        ├── 19_요기_몸과움직임/
        ├── 20_숨이_트라우마와안정화/
        └── 21_측정도구_평가척도/
```

## 제외한 것

- 원본 `자료/` 전체
- 원본 백업 zip
- 브랜드 캠페인 전체
- 블로그 제작 과정 전체
- 자동수집/기타 실험 폴더

위 항목은 서비스 구현과 캐릭터 GPT 전환에 직접 필요한 핵심은 아니므로 제외했다. 필요하면 원본 `10-일상다반사/`에서 추가로 가져온다.

## 현재 프로젝트 한 줄

`보살피고`: 마음·모미·요기·숨이 4 캐릭터가 몸, 마음, 움직임, 안정화 영역을 나누어 사용자의 자기 인식을 돕는 한국어 대화형 멘탈 컴패니언.

## 주요 경로

- 서비스 계획/로드맵: `마음_프로덕트/01_service/`
- 캐릭터 버전 지침: `마음_프로덕트/04_캐릭터_버전별_지침/`
- 실제 웹앱 소스: `outputs/마음_앱/`
- v1.5 베타 앱: `outputs/마음_앱/v1.5/`
- GPT API 버전 앱: `outputs/보고살피다_GPT/`
- GitHub Pages 배포본: `docs/index.html`
- 요기/숨이/측정도구 학습자료: `자료/04_캐릭터_학습자료/`

## 실행

정적 웹앱이므로 별도 빌드가 필요 없다.

```bash

```

